# Employee Management System 💼

A Java-based Employee Management System that allows users to manage employee records including adding, updating, viewing, and removing employees.

## 🚀 Features
- Add Employee  
- View Employee  
- Update Employee  
- Delete Employee  
- Login System  

## 🛠️ Tech Stack
- Java  
- AWT / Swing  
- JDBC (if using database)

## ▶️ How to Run
```bash
javac *.java
java Login
